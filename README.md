# Ananya Vig - Portfolio

A modern, responsive portfolio website showcasing full-stack development projects with smooth parallax animations.

## 🚀 Features

- **Smooth Parallax Animations** - Multi-layer scroll effects using Framer Motion
- **Responsive Design** - Works seamlessly on all devices
- **Smooth Scrolling** - Powered by Lenis for buttery smooth scroll experience
- **Modern Stack** - Built with React and Vite

## 🛠️ Tech Stack

- **React** - UI framework
- **Vite** - Build tool and dev server
- **Framer Motion** - Animation library
- **Lenis** - Smooth scroll library
- **Wouter** - Lightweight routing

## 📦 Installation

```bash
npm install
```

## 🏃‍♂️ Running Locally

```bash
npm run dev
```

Open [http://localhost:5000](http://localhost:5000) in your browser.

## 🔨 Building for Production

```bash
npm run build
```

The build output will be in the `dist` folder.
