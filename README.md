
# Ananya Vig - Portfolio

A modern, visually engaging portfolio website to showcase selected projects, skills, education, certifications, and contact information. Built with React, Vite, and Framer Motion, featuring smooth parallax animations and a custom Apple-style dock navigation.

## 🚀 Features

- **Apple-style Dock Navbar** with animated tooltips and smooth section scroll
- **Multi-layer Parallax Animations** using Framer Motion
- **Responsive Design** for all devices
- **Smooth Scrolling** powered by Lenis
- **Project Gallery** with animated cards and GitHub links
- **Downloadable Resume** and direct contact links
- **Modern UI/UX** with custom cursor and loading animation

## 🛠️ Tech Stack

- **React 18** – UI framework
- **Vite** – Build tool and dev server
- **Framer Motion** – Animation library
- **Lenis** – Smooth scroll library
- **Lucide React** – Icon set
- **Wouter** – Lightweight routing (minimal usage)

## 📦 Installation

```bash
npm install
```

## 🏃‍♂️ Running Locally

```bash
npm run dev
```

Open [http://localhost:5000](http://localhost:5000) in your browser.

## 🔨 Building for Production

```bash
npm run build
```

The build output will be in the `dist` folder.
